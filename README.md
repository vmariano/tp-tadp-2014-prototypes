tp-tadp-2014-prototypes
=======================
##Setup

####Install rvm

```bash 
   rvm install 2.1.2
```

####force login over bash (if you use other shells like zsh)

```bash 
    /bin/bash --login
```

####Set current vm version

```bash
    rvm use 2.1.2
```
####Install Rspec gem.

```bash 
    gem install rspec
```
